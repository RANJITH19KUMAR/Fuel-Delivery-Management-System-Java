# Fuel Delivery Management System (Java + MySQL)

A console-based backend application built using Core Java and JDBC, 
simulating a real-world fuel ordering and delivery system.

## Features
- User registration & login
- Fuel selection & order placement
- Payment handling (UPI / Card / Cash)
- Delivery address confirmation
- OTP-based delivery verification
- Admin panel to monitor all orders

## Tech Stack
- Java (Core Java, JDBC)
- MySQL
- IntelliJ IDEA

## Architecture
- DAO Pattern
- Separate modules for:
  - Login
  - Orders
  - Payments
  - Delivery
  - Admin

## Admin View
Displays:
- Order details
- User information
- Payment status
- Delivery status
